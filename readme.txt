thinkery.me implemented as a WordPress plugin.

== Description ==

This plugin can be the future, self-hosted home of your thinkery.

== Installation ==

1. Upload the `thinkery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Potentially import your thinkery.me data under 'Settings' > 'Import'
